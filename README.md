# Adivansh ATC System ✈️

Adivansh is a high-performance, real-time Air Traffic Control (ATC) management system designed to simulate and optimize airport operations. This system was specifically tailored and optimized for **Kempegowda International Airport (BLR)**, focusing on realistic flight physics and automated runway management.

##  Key Features

- **Advanced Departure Management**: Automatically handles flight initialization from ground level (0 altitude).
- **Realistic Flight Physics**: Implements a climbing simulation that calculates altitude and velocity based on takeoff parameters.
- **Automated Runway Logistics**: Features an automated runway clearance reset system to ensure continuous, high-volume traffic flow without manual intervention.
- **Real-time Dashboard**: A modern, responsive interface built with React to monitor multiple flight vectors simultaneously.
- **BLR Optimized**: Logic tuned for the specific operational patterns of Bangalore International Airport.

##  Tech Stack

- **Frontend**: [React.js](https://reactjs.org/)
- **Bundler**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Logic**: Custom JavaScript flight physics engine
